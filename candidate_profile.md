## Autonmis Internship
During my internship at Autonmis, I spearheaded the integration of the Alert Report AI system. My primary focus was building a comprehensive RAG (Retrieval-Augmented Generation) knowledge base that allowed the chatbot to instantly surface critical incident reports and documentation to the operational team. I developed the frontend dashboard using React and Next.js to display real-time insights derived from the AI analysis. The chatbot work significantly reduced the time it took for the support team to find legacy alert resolutions, improving response times by over 40%.

## AI Recruiter Project
I built a fully autonomous AI Recruiter platform designed to conduct initial screening interviews. The architecture utilized Next.js for the frontend, a Node.js backend, and integrated deeply with OpenAI's API and Supabase for vector search (RAG). The system was capable of analyzing a candidate's resume, asking tailored technical questions based on their experience, and autonomously scheduling follow-up interviews using calendar tool calling. 

## IoT Project: Smart Environment Monitor
I developed a Smart Environment Monitor using a Raspberry Pi and a suite of IoT sensors (temperature, humidity, CO2). The device collected data locally and streamed it via MQTT to a cloud backend built on AWS (IoT Core and DynamoDB). I created a real-time visualization dashboard using React and Recharts that allowed users to monitor air quality trends and receive automated alerts via Twilio when CO2 levels exceeded safe thresholds. 

## Alzheimer's Project
I contributed to a research-driven software initiative focused on early detection of Alzheimer's through speech pattern analysis. I trained a machine learning model using PyTorch on a dataset of recorded patient interviews to identify subtle cognitive decline markers (like frequent pausing or vocabulary simplification). The backend was deployed via a Python FastAPI service, and I built a cross-platform mobile application using React Native to allow clinicians to record and analyze patient speech in real-time securely.
